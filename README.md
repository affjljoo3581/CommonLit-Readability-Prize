# CommonLit Readability Prize

## Introduction
This repository is the code that placed 42nd in [CommonLit Readability Prize](https://www.kaggle.com/c/commonlitreadabilityprize) competition.

## Requirements
* transformers==4.8.2
* optuna
* omegaconf
* numpy
* wandb
* pandas
* sentencepiece
* torch==1.8.1+cu111
* pytorch_lightning==1.3.8
* scikit_learn
